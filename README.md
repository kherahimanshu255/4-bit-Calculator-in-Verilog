# 4-bit-Calculator-in-Verilog
<p>Specifications:</p>
•	Inputs: 2 x 4-bit (Data), 1 x 2-bit (Opcode). <br>
•	Output: 1 x 5-bit (Result). <br><br>
<p>Operation Table: </p>

| Op (Binary) | Function | Logic |
| :--- | :--- | :--- |
| 00 | Addition | Result = A + B |
| 01 | Subtraction | Result = A - B |
| 10 | AND | Result = A & B |
| 11 | OR | Result = A | B |
